# FAQ Chatbot

An intelligent FAQ Chatbot built with Streamlit and NLP techniques using TF-IDF Vectorization and Cosine Similarity.

## Features
- AI & Data Science FAQ responses
- NLP preprocessing using NLTK (stopwords removal, tokenization)
- TF-IDF Vectorization for text representation
- Cosine Similarity for finding best matching answer
- Simple and clean chat UI

## Topics Covered
Artificial Intelligence, Machine Learning, Deep Learning, NLP, Computer Vision, Neural Networks, Data Science, Python, Chatbots, Reinforcement Learning

## Installation

pip install -r requirements.txt

## Run the App

streamlit run app.py
